# nginx 安装
下载 nginx 安装包：[1.18.0安装包](https://nginx.org/download/nginx-1.18.0.tar.gz)
```shell
# 下载 nginx 源码包
wget https://nginx.org/download/nginx-1.18.0.tar.gz
# 解压压缩包
tar -zxvf nginx-1.18.0.tar.gz
# 预先安装软件(注意我是在 root 下安装的，所以不需要 sudo 命令)
# 安装 nginx 需要先将官网下载的源码进行编译，编译依赖 gcc 环境，如果没有 gcc 环境，则需要安装：
yum install gcc-c++
# PCRE(Perl Compatible Regular Expressions) 是一个Perl库，包括 perl 兼容的正则表达式库。nginx 的 http 模块使用 pcre 来解析正则表达式，所以需要在 linux 上安装 pcre 库，pcre-devel 是使用 pcre 开发的一个二次开发库。nginx也需要此库。命令：
yum install -y pcre pcre-devel
# zlib 库提供了很多种压缩和解压缩的方式， nginx 使用 zlib 对 http 包的内容进行 gzip ，所以需要在 Centos 上安装 zlib 库。
yum install -y zlib zlib-devel
# 安全套接字层密码库，用于通信加密
yum -y install openssl openssl-devel
# 也可以一键安装上述四个软件
yum -y install gcc pcre-devel zlib-devel openssl openssl-devel
# 设置安装目录
./configure --prefix=/usr/local/nginx # 指定安装目录
# 进行编译并安装
make && make install
```
安装完成之后，配置环境

```shell
vim ~/.bash_profile 

# nginx config
export NGINX="/usr/local/nginx/sbin"

PATH=$PATH:$HOME/bin:$NGINX
```

配置代理接口(先备份一下 nginx 的配置目录)：
```shell
# 切换到上述编译生成的目录
cd /usr/local/nginx/
# 配置文件在 conf 文件夹下
cp nginx.conf nginx.conf_back
# 开始配置 nginx 访问接口
vim nginx.conf

    server {
        listen       80;
        server_name  10.211.55.3;
        
# 启动 nginx 服务，在这之前直接把防火墙关掉，自行百度
nginx
# 停止 nginx
nginx -s stop
# 重新加载 nginx
nginx -s reload
# 查看 nginx 版本
nginx -v  
```
